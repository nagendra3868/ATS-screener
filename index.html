About Project

Smart ATS Resume Keyword Screener is a lightweight Applicant Tracking System built using HTML, CSS, JavaScript, and Python. It reads resume text, compares it with predefined technical keywords stored in CSV files, calculates an ATS match score, and classifies the resume as a Strong Match, Partial Match, or Needs Review.
